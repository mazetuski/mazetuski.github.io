Listado JavaScript

Tres argumentos de forEach:

  • currentValue: el elemento siendo procesado por el array.
  • index: el indice del elemento actual
  • array: el array en el que el forEach se esta usando

Operador in: devuelve true si la propiedad especificada esta dentro del array, (elem in array)

La función que comprueba si un elemento es un array o no: Array.isArray('array a comprobar');

Diferencias entre los metodos:

  • Array.prototype.forEach: comprueba que cada uno de los elementos cumplen la función que se le pasa.
  • Array.prototype.every: comprueba que todos los elementos del array cumplen la función que se le pasa como argumento.
  • Array.prototype.some: comprueba que uno de los elementos del array cumple la función que se le pasa como argumento.
  • Array.prototype.filter: comprueba que cada uno de los elementos del array cumplen la función que se le pasa como argumento.
